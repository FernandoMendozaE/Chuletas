* Guía para crear una página para un repositorio GitHub
* Comandos probados sólo en Debian



// ------------------------------ Guía para páginas en Git [BEGIN] ------------------------------ //
1. Crear un repositorio en GitHub llamado username.github.io cambiando username por el nombre de
usuario que se posee en GitHub.

2. Clonar el repositorio:
  $ git clone https://github.com/username/username

3. Entrar en el directorio que se clonó
  $ cd username.github.io

4. Crear un archivo index.html:
  $ echo "Hello World" > index.html

5. Agregue, comente y haga push de los cambios:
  $ git add --all
  $ git commit -m "Commit Inicial"
  $ git push -u origin master

6. Con el navegador de su preferencia entre a username.github.io
